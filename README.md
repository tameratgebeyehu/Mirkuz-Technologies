## 🌟 About the Portfolio
This portfolio is engineered to be a premium, app-like experience on the web. It features:
- **Zero-Trust Security Focus:** Highlighting my commitment to E2E encryption and enterprise-grade architecture.
- **Fluid UI & Animations:** Powered by `framer-motion` for a physics-based, interactive aesthetic.
- **Glassmorphic Design System:** A stunning visual language using custom dark modes and dynamic gradients.
- **Direct APK Distribution:** Integrated system to serve production-ready `.apk` binaries directly from the repository.

## 🛠 Core Tech Stack
- **Frontend Framework:** React 18 with Vite for lightning-fast HMR.
- **Routing:** React Router v6.
- **Animations:** Framer Motion & pure CSS micro-interactions.
- **Icons:** Lucide React.
- **Styling:** Vanilla CSS with a bespoke token system (`portfolioData.js`).
- **Deployment:** Vercel (CI/CD).

## 🚀 Getting Started Locally

Want to run this portfolio on your local machine?

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tameratgebeyehu/Mirkuz-Technologies.git
   cd Mirkuz-Technologies
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in Browser:**
   Open `http://localhost:5173` in your browser.
